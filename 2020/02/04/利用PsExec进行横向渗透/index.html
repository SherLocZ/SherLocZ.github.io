利用PsExec进行横向渗透

发表于 2020-02-04

利用条件

  • 需要目标机器开启了ADMIN$共享(默认开启)
  • 建立了IPC连接，可以不输入账号密码连接到CMD窗口

下载地址:https://docs.microsoft.com/zh-cn/sysinternals/downloads/pstools

优点：可直接获取SYSTEM权限的交互式shell
缺点：利用时会产生一个Psecex的服务，这个服务会产生日志文件，容易溯源

模拟利用过程

  • Win Server 2012 R2(DC)
  • Win 7(域中的一台机器)
